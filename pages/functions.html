Functions

Functions are nice little blocks of code that can be reused without retyping everything. They are methods of manipulating values. Functions are often used to perform a calculation and return the result, or simply calculating something and changing the value automatically.

For example, here is some quick code that would return a result:


def area(length):
    return length * length

print('The area of a 2x2 square is ' + area(2))
    # Result: "The area of a 2x2 square is 4"

Next, some code that alters an already existing value:


changeme = "value 1"
print(changeme)
# Result: "value 1"

def new_message(msg):
    changeme = msg

print(changeme)
# Result: "value 1"

new_message("hello!")
print(changeme)
# Result: "hello!"

As you can see in the last example, the code inside of a function only happens when that function is called.

KEEP IN MIND! When you call a function, it runs! That seems self explanatory, but if you're not careful you could run into a situation where it becomes an issue.

Take this situation for example: You have a function that generates a random message. You want to use that message in multiple places, so you try the following:


import random
def rand_msg():
    msgs = ["hello", "goodbye", "have a nice day"]
    choice = random.randint(0,len(msgs)-1) # Takes a random number between 0 and 2, the range of indexes of the msgs list
    return msgs[choice]

print('Msg 1: ' + rand_msg())
print('The same msg: '+ rand_msg())

If you ran this code multiple times, you would notice that the two print statements wouldn't always return the same message. That is because each time you use rand_msg(), the function runs again and a new random message is returned.

To get around this, you can set function results as variables. That feels like a mouthful, but I promise it's not complicated. Editing the code from above, we get:


import random
def rand_msg():
    msgs = ["hello", "goodbye", "have a nice day"]
    choice = random.randint(0,len(msgs)-1) # Takes a random number between 0 and 2, the range of indexes of the msgs list
    return msgs[choice]

message = rand_msg()
print('Msg 1: ' + message)
print('The same msg: '+ message)

When you use a variable, the function only runs once: when you first assign the variable.
